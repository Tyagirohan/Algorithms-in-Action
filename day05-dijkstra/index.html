What is Dijkstra's?

A greedy algorithm that finds the shortest path from a source node to all other nodes in a weighted graph with non-negative edges.

ğŸ”

How It Works

Always visit the unvisited node with the smallest known distance. Update distances to neighbors, then mark as visited. Repeat until destination is reached.

âš¡

Time Complexity

O((V + E) log V) with priority queue, where V = vertices and E = edges. Efficient for sparse graphs!

âœ…

Why It Works

Greedy choice property: Once a node is visited, we've found its shortest path. No shorter path exists through unvisited nodes.

ğŸ“ Algorithm Steps

1

Initialize

Set distance to source = 0, all others = âˆ. Mark all nodes as unvisited.

2

Select Minimum

Choose the unvisited node with the smallest distance (greedy choice).

3

Update Neighbors

For each neighbor, calculate: distance[current] + edge_weight. If smaller than current distance, update it.

4

Mark Visited

Mark the current node as visited. It will not be checked again.

5

Repeat

Repeat steps 2-4 until destination is visited or all reachable nodes are visited.

function dijkstra(graph, start, end) {
    const distances = {};
    const visited = new Set();
    const previous = {};
    const pq = new PriorityQueue();
    
    // Initialize distances
    for (let node in graph) {
        distances[node] = Infinity;
    }
    distances[start] = 0;
    pq.enqueue(start, 0);
    
    while (!pq.isEmpty()) {
        const current = pq.dequeue();
        
        if (visited.has(current)) continue;
        if (current === end) break;
        
        visited.add(current);
        
        // Check all neighbors
        for (let neighbor of graph[current]) {
            if (visited.has(neighbor.node)) continue;
            
            const newDist = distances[current] + neighbor.weight;
            
            if (newDist < distances[neighbor.node]) {
                distances[neighbor.node] = newDist;
                previous[neighbor.node] = current;
                pq.enqueue(neighbor.node, newDist);
            }
        }
    }
    
    // Reconstruct path
    const path = [];
    let current = end;
    while (current) {
        path.unshift(current);
        current = previous[current];
    }
    
    return { distance: distances[end], path };
}
